A simple recipe app to help you record and organize your favorite recipes!  

## Features  
- User login
- User Sign Up
- Recipe display  
- Add new recipes
- Edit recipes

## Tech Stack  
- **React Native**  
- **Firebase**  

## Installation  

1. Clone the repository:  
   git clone <repository_url>
   
Navigate to the project directory:
cd <project_directory> 

Install dependencies:
npm install  

Run the app:
npm start
